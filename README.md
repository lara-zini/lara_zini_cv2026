El trabajo práctico se basó en la realización de mi Curriculum Vitae, para el mismo trabajé con el editor de código Visual Studio y un monitor de 1920x1080px.

Comencé realizando un boceto manual del orden del contenido que quería colocar. 
Luego, fui recopilando la información y colocándola en el archivo HTML5 (/index) con el orden y la jerarquía deseada. 

Previo a darle propiedades CSS a esas etiquetas hice un modelo de boceto en Adobe Illustrator para facilitar la visualización del resultado final buscado. En este boceto organicé el espacio compositivo del contenido, seleccioné tipografías y formé la paleta cromática.
Una vez realizado el boceto hice el archivo CSS (/estilos) y comencé a dar propiedades.

También necesité crear una carpeta para colocar las tipografías (/fonts), las cuales también tuve que linkear al CSS, y otra para las imágenes (/img), las cuales edité con Adobe Photoshop para darles el tamaño y resolución correcto.
Fui probando variantes con el inspector de Google Chrome y luego lo pasé a mis archivos. Este proceso me llevó un tiempo, lo repetí varios días hasta modificar los últimos detalles y llegar al resultado final que buscaba.

Una vez que obtuve el resultado deseado sincronicé las modificaciones finales a mi repositorio de Github y realicé la entrega.
